make_path

make_path(opts)

設定されているファイルパスのフォルダを作ります。

mkdir -p を使ってつくります。失敗したら missing valueが返ります。引数にオプションとしてレコードをあたえ動作をカスタマイズできます。

{with_admin : boolean}

今のところ、with_admin だけです。true を、与えると管理者権限でフォルダを作ります。オプションを指定しない場合は、空のリスト {} を与えてください。

Result

script object or missing value : 新しく作ったフォルダの XFile インスタンス。
