XFile Reference Edit Script
unique_child

unique_child(a_candidate)

フォルダ内で他と名前が重複しない XFile インスタンスを生成します。

フォルダでない XFile のインスタンスに対して実行すると、missing value が返ります。XFile のインスタンスがファイル参照が存在しなければ、error number 1350 が発生します。

Parameters
  • a_candidate(Unicode text or list) : 取得したいファイル参照の名前の候補。既に存在する名前であれば変更が加えられます。
Result

script object : XFile インスタンス
